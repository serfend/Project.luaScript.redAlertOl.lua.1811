# 使用方法

###模拟器

1. 下载并安装模拟器，推荐使用夜神模拟器。
2. 设置分辨率为`1080*1920:480dpi` 或`720*1280:320dpi`
3. 模拟器可免root运行叉叉助手和小精灵，免去电脑激活的步骤

###叉叉助手

1. 百度搜索叉叉助手，下载平台
2. 搜索`红警ol`，找到此脚本进行运行
